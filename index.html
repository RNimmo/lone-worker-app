How to Use This App

This app helps ensure the safety of lone workers by logging visits and sending automated alerts if a worker is overdue.

1. Initial Setup (Crucial)

Go to Settings and fill in ALL fields. You MUST provide a Google Sheet Web App URL for data logging. Without it, the app cannot record your visits.

2. Setting up Google Sheets

  1. Create a new Google Sheet. Add these headers in the first row: `Date`, `LocationName`, `LocationAddress`, `ArrivalTime`, `ActualDepartureTime`, `AnticipatedDepartureTime`, `AlarmStatus`, `WorkerName`, `WorkerPhoneNumber`, `EmergencyContact`, `EmergencyContactNumber`, `Notes`.
  2. In the Sheet, go to `Extensions > Apps Script`.
  3. Delete any existing code and paste the code provided below.
  4. Click the "Save project" icon.
  5. Click the "Deploy" button, then "New deployment".
  6. For "Select type", choose "Web app".
  7. Under "Configuration", set "Execute as" to "Me".
  8. Crucially, set **"Who has access" to "Anyone"**. This does NOT make your sheet public, it only allows this script to add rows.
  9. Click "Deploy". You will need to "Authorize access" with your Google account. Click through the advanced/unsafe warnings.
  10. Copy the generated "Web app URL" and paste it into the Settings page of this app.

Google Apps Script Code:

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0];
    var data = JSON.parse(e.postData.contents);
    
    // Find a row to update or append a new one
    var lastRow = sheet.getLastRow();
    var arrivalTimes = sheet.getRange(2, 4, lastRow > 1 ? lastRow - 1 : 1, 1).getValues();
    var workerNames = sheet.getRange(2, 8, lastRow > 1 ? lastRow - 1 : 1, 1).getValues();
    
    var rowToUpdate = -1;
    // Search backwards for the latest entry for this worker
    for (var i = arrivalTimes.length - 1; i >= 0; i--) {
      if (arrivalTimes[i][0] == data.ArrivalTime && workerNames[i][0] == data.WorkerName) {
        rowToUpdate = i + 2; // +2 because sheets are 1-indexed and we skipped header
        break;
      }
    }

    if (rowToUpdate != -1) {
      // Update existing row
      var range = sheet.getRange(rowToUpdate, 1, 1, 12);
      var currentValues = range.getValues()[0];
      var newValues = [
        data.Date || currentValues[0],
        data.LocationName || currentValues[1],
        data.LocationAddress || currentValues[2],
        data.ArrivalTime || currentValues[3],
        data.ActualDepartureTime || currentValues[4],
        data.AnticipatedDepartureTime || currentValues[5],
        data.AlarmStatus || currentValues[6],
        data.WorkerName || currentValues[7],
        data.WorkerPhoneNumber || currentValues[8],
        data.EmergencyContact || currentValues[9],
        data.EmergencyContactNumber || currentValues[10],
        data.Notes ? (currentValues[11] ? currentValues[11] + '; ' + data.Notes : data.Notes) : currentValues[11]
      ];
      range.setValues([newValues]);

    } else {
      // Append new row
      sheet.appendRow([
        data.Date, data.LocationName, data.LocationAddress,
        data.ArrivalTime, data.ActualDepartureTime, data.AnticipatedDepartureTime,
        data.AlarmStatus, data.WorkerName, data.WorkerPhoneNumber,
        data.EmergencyContact, data.EmergencyContactNumber, data.Notes || ""
      ]);
    }

    return ContentService.createTextOutput(JSON.stringify({ "status": "success", "row": rowToUpdate }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

Important Note on SMS

For security reasons, this web app cannot send SMS messages automatically. When an alert is triggered, it will open your phone's default messaging app with the message pre-filled. **You must press 'Send' yourself.**
